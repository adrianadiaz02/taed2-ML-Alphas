# TAED2-ML-Alphas

## Alzheimer MRI Disease 

## Dataset Card
This dataset focuses on the classification of Alzheimer's disease based on MRI scans. The dataset consists of brain MRI images labeled into four categories:
'0': Mild_Demented
'1': Moderate_Demented
'2': Non_Demented
'3': Very_Mild_Demented
